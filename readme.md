## Penjelasan Program Hitung Luas dan Keliling Lingkaran menggunakan method

cara kerja programnya:
1. ada sebuah object bernama ling
2. selanjutnya didalamnya terdapat 3 method yang bernama luas, kel, dan hitung
3. dalam metod luas dan kel itu ada fungsi dengan parameter r yang mana itu merupakan jari jari dan parameter phi yang merupakan nilai dari phi. dimana kedua method tersebut pada fungsinya berisi rumus aritmatika untuk menghitung luas dan keliling lingkaran. dimana pada akhir masing masing fungsi ada perintah return untuk mengembalikan nilai dari perhitungan aritmatikanya
4. selanjutnya ada metod hitung, dimana pada fungsinya ada parameter r yang merupakan nilai jari jari, fungsi ini berisi proses untuk mengecek apakah r itu merupakan number atau bukan, menentukan nilai dari phi dan untuk mencetak hasil luas dan keliling lingkaran berdasarkan method dari luas dan kel.
5. terakhir untuk memunculkan programnya methodnya dipanggil beserta fungsinya, dan diberikan parameter jari jari atau r
